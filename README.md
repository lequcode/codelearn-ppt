# CodeLearn 产品介绍

在线演示：https://lequ-edu.github.io/codelearn-ppt

## 关于

CodeLearn 编程教学系统产品介绍网页PPT，基于 reveal.js 构建。

## 快速预览

直接在浏览器中打开 `index.html` 即可查看。

或使用本地服务器：

```bash
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 内容

- 12页完整产品介绍
- 包含痛点分析、解决方案、功能介绍、定价方案、FAQ
- 响应式设计，支持键盘导航

## 技术栈

- reveal.js 4.5.0
- HTML5 + CSS3
